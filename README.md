# FinanceOS v6.5

Executive Financial Management PWA

## Features

- 🔐 PIN Security (4-digit vault)
- 💰 Net Worth Dashboard
- 📊 Account Tracking
- 📝 Transactions with Auto-Categorization
- 👁️ Privacy Blur Toggle
- 🔍 Search & Filters
- 💾 JSON Export/Backup
- 📱 Works as Native App

## Quick Start

1. Open `index.html` in your browser
2. Create a 4-digit PIN
3. Add transactions
4. All data saves locally on your device

## Installation

**GitHub Pages:**
1. Fork this repo
2. Enable Pages in Settings
3. Visit `https://YOUR_USERNAME.github.io/FinanceOS`

**Local:**
- Just open `index.html` in any browser

## Features

- **Dark OLED Theme** - Easy on the eyes
- **Gold Accents** - Premium aesthetic
- **Expandable Details** - Click transactions to see Authority ID & notes
- **Real-time Liquidity** - Shows savings rate %
- **Portfolio Status** - Elite/Stable/Aggressive badges
- **No Cloud** - All data stays on your device

## No Installation Required

Single HTML file - no npm, no build tools, no dependencies.
